## Mapa województw
#### Informacje
Interaktywna mapa, która prezentuje informacje dotyczące poszczególnych województw.
	
#### Technologie
* React
* ES6
* Webpack
* Sass
* svg_map: https://www.amcharts.com/svg-maps/?map=world
	
#### Instalacja
```
$ npm i
$ npm start
```

#### Wersja DEMO:
https://wojciechkkk.pl/map
