## Mapa województw
#### Informacje
Strona prezentuje informacje dot. poszczególnych województw.
	
#### Technologie
* React
* ES6
* Webpack
* Sass
* SVG: https://www.amcharts.com/svg-maps/?map=world
	
#### Instalacje
- przejdź do folderu 'app', nastepnie uruchom polecenia:
```
$ npm i
$ npm start
```
